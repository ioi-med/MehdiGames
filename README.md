# MehdiGames : Le Retour de Stalum

**Créé par MehdiLabs**

Bienvenue dans **MehdiGames**, un Action-Platformer 2D développé intégralement en HTML5, CSS3, et Vanilla JavaScript (sans aucun moteur externe ni assets pré-calculés).

Le jeu utilise un système unique de rendu **Vrai Pixel Art Dynamique**, générant 100% de ses ressources visuelles (Slimes, Squelettes, Héros, UI) mathématiquement via des matrices de pixels au chargement.

## 🚀 Fonctionnalités

- **100 Niveaux Procéduraux** : De la plateforme intense, des pièges mortels et des nuées d'ennemis.
- **10 Boss Épiques** : Un boss massif tous les 10 niveaux, avec des paterns d'attaque spécifiques.
- **Support Multi-Plateforme** : Jouable sur PC, Mobile (iOS/Android) et Tablette.
- **2 Modes de Jeu** :
  - **Survie** : Une aventure difficile avec points de vie, où le vide et les monstres ne pardonnent pas.
  - **Créatif** : Vous êtes un dieu immortel. Explorez les niveaux sans subir de dégâts.
- **Sauvegarde Intégrée** : Sauvegardez manuellement vos prouesses au format `.json` depuis le menu Pause, et reprenez là où vous vous étiez arrêté.

## 🎮 Contrôles & Manette (Gamepad API)

Le jeu détecte automatiquement le clavier, les écrans tactiles et les **manettes de jeu (PlayStation, Xbox, génériques)** connectées en USB ou Bluetooth.

| Action | Clavier (PC) | Manette (Gamepad) | Tactile (Mobile) |
| :--- | :--- | :--- | :--- |
| **Se Déplacer** | Flèches ⬅️➡️ ou `A`/`D` / `Q`/`D` | Croix directionnelle / Joystick Gauche | Boutons virtuels ◄ / ► |
| **Sauter** | Flèche ⬆️ ou `W` / `Z` | Bouton A (Croix PS) | Bouton virtuel ▲ |
| **Attaquer** | Barre ESPACE ou `X` | Bouton X (Carré PS) ou B | Bouton virtuel ⚔ |
| **Pause / Menu** | `ECHAP` ou `P` | Bouton Start / Options | N/A |

> *Astuce Menu : Vous pouvez naviguer dans les menus avec les directions, et valider avec Entrée, Espace ou le Bouton A de votre manette.*

## 🔧 Installation & Lancement

Rien à compiler, rien à héberger sur un serveur complexe.
1. Ouvrez le dossier du projet.
2. Double-cliquez sur `index.html`.
3. Le jeu se lance instantanément dans votre navigateur par défaut.

Idéal pour un hébergement statique sur **GitHub Pages**, Vercel ou Netlify.
